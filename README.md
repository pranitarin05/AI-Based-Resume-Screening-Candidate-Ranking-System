# AI-Based-Resume-Screening-Candidate-Ranking-System
AI-Based Resume Screening System that ranks resumes against a job description using NLP techniques. Extracts text from PDF/DOCX files, applies preprocessing, converts text using TF-IDF, and calculates cosine similarity to generate match scores and rank candidates automatically.
